# 🥊 Boxing Club — Projeto de Portfólio

Site completo de uma academia de boxe, desenvolvido por **Rafael Vetrano** como projeto de portfólio para vaga de desenvolvedor júnior.

🔗 **Acesse a demo:** `<URL do Cloudflare Pages aqui>`

---

## O que é este projeto

Um site real de academia, com login, cadastro, área do aluno, loja com carrinho de compras, planos de assinatura e painel administrativo — como um sistema completo que uma academia usaria de verdade no dia a dia, não apenas uma página de apresentação.

## O que dá para testar na demo

- Criar uma conta e confirmar por e-mail
- Ver os horários de aula
- Ver os planos disponíveis
- Navegar pela loja de produtos e adicionar itens ao carrinho
- Acessar a área do aluno (perfil, pedidos)
- Painel de administrador, com gestão de horários, produtos, pedidos e usuários

## Único ponto desativado de propósito

O **pagamento** (cartão, PIX, boleto) está desativado nesta demo. Isso é intencional: o pagamento usa uma integração real com o Mercado Pago, que exige credenciais financeiras — não faz sentido deixar isso ativo numa demonstração pública. Todo o resto do site funciona normalmente.

## Tecnologias utilizadas

React, TypeScript, Node.js, PostgreSQL — com banco de dados, login seguro e e-mails automáticos de verdade (não é só "de fachada").
